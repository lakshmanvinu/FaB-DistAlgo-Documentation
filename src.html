checkCorrectness()¶
rule_cl1()¶

Some proposed value is eventually chosen.

rule_cl2()¶

Once a value is chosen, correct learners eventually learn the value.

rule_cs1()¶

Only a value that has been proposed must be chosen

rule_cs2()¶

Only a single value may be chosen. For every proposal number atmost one value is chosen.

rule_cs3()¶

Only Chosen value may be learned by a correct learner. Some value is eventually stable.

onPropose(leader, value, pnumber, progcert)¶

Receive Proposal from a leader * If the value is proposed by the elected leader, only then the acceptor listens to the proposer, else discard the message * If a new value is proposed by the leader, then change the progress certificate. * Else, accept the value sent by the leader and send accepted message to all the learners and acceptors about the learned value.

Parameters:
  • leader (Process) – The leader process
  • value (int) – The value accepted
  • pnumber (int) – The proposal number of the accepted value
  • progcert (List<tuples>) – The progress certificate is a list of valid (proposal numbers, accepted values)
